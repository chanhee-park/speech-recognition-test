About Speech Synthesis

The Speech Synthesis interface of the Web Speech API is responsible for speech synthesis (also known as text-to-speech or TTS). It allows you to convert text into spoken words.

The SpeechSynthesis interface of the Web Speech API is the controller interface for the speech service; this can be used to retrieve information about the synthesis voices available on the device, start and pause speech, and other commands besides. Learn more about Speech Synthesis

You can access the source code for this project on GitHub: speech-recognition-test.
